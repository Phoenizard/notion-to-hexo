# Code of Conduct

This project follows the Contributor Covenant 2.1. We expect all contributors to:
- Be respectful and considerate in interactions
- Avoid harassment, intimidation, and discriminatory language
- Be open to constructive feedback

Reporting
If you experience or witness unacceptable behavior, please report it to the project maintainers by opening a private issue or using GitHub's reporting tools. Maintainers will respond and take action as appropriate.

Consequences
Project maintainers may remove, reject, or ask to modify content or contributions that violate this Code of Conduct.
